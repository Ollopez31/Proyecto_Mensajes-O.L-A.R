# Proyecto_Mensajes-O.L-A.R
#Introduccion
En la actualidad, la comunicación digital es fundamental en la vida diaria. Aplicaciones como WhatsApp permiten la interacción inmediata entre usuarios.
Este proyecto propone el desarrollo de un sistema de chat que simula el envío y recepción de mensajes, aplicando los principios de la programación orientada a objetos.
#Objetivo del proyecto
Desarrollar un sistema de chat que permita la comunicación entre usuarios mediante el envío y recepción de mensajes, aplicando principios de la programación orientada a objetos para garantizar una estructura  organizada y escalable.
#Clases del sistema
1) Usuario
2) Mensaje
3) Chat
#Paquetes
Net => Conexion TCP entre clientes
Bufio => Escritura i/o de texto
sync => Sirve para el control de concurrencia
